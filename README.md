1. How to use?

```
import { CalendarWrapper } from 'solid-calendar'
```

2. Create a signal with empty object

```
 const [date, setDate] = createSignal({})
```

3. EZ

```
<CalendarWrapper type="form" setDates={setDate} dates={date} />
```
